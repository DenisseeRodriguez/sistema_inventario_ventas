El siguiente proyecto hace refencia al desarrollo del siguiente sistema de Inventario y Ventas.

Este sistema tendra los modulos de compras y ventas, y por ello entonces se podra administrar el inventario por medio de un control de inventario. Dicha gestion de inventarlo administra las entradas de productos (compras) salida de productos (ventas) y se podra generar reportes de inventario por stock

Sistema desarrollado por el aula 3DS
